## 🚀 How to Use

### Desktop App (Windows)
```
1. Navigate to: stack-family-finance\release\win-unpacked\
2. Double-click: STACK Kids Bank.exe
3. Done!
```

### Android App
```
1. Copy STACK-Kids-Bank.apk to your phone
2. Open the file on your phone
3. Tap "Install"
4. Allow "Unknown Sources" if prompted
5. Open "STACK Kids Bank"
```

### Web App (Local)
```powershell
# Backend
cd stack
mvn spring-boot:run

# Frontend
cd stack-family-finance
npm run dev

# Open: http://localhost:8081
```

### Web App (Deploy Online)
```
1. Go to netlify.com or vercel.com
2. Sign up (free)
3. Drag stack-family-finance\dist folder
4. Get URL and share!
```
