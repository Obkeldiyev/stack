# 🚀 Quick Start Guide

## Start Backend
```bash
cd stack
mvn spring-boot:run
```
Backend runs on: http://localhost:8080

## Start Frontend (Web)
```bash
cd stack-family-finance
npm run dev
```
Frontend runs on: http://localhost:8081

## Or Use Desktop App
Double-click: `STACK-Kids-Bank-Setup-FIXED.exe`

Or run directly: `stack-family-finance\release\win-unpacked\STACK Kids Bank.exe`
