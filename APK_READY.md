## 📱 How to Install on Your Phone

### Method 1: USB Cable (Easiest)

1. **Connect your phone to computer via USB**

2. **Copy APK to phone:**
   - Open File Explorer
   - Find your phone in "This PC"
   - Copy `STACK-Kids-Bank.apk` to your phone's Downloads folder

3. **On your phone:**
   - Open "Files" or "My Files" app
   - Go to Downloads folder
   - Tap on `STACK-Kids-Bank.apk`
   - Tap "Install"
   - If prompted, allow "Install from Unknown Sources"
   - Tap "Open" when installation completes

### Method 2: Email/Cloud

1. **Email the APK to yourself** or upload to Google Drive/Dropbox

2. **On your phone:**
   - Open the email or cloud app
   - Download `STACK-Kids-Bank.apk`
   - Tap the downloaded file
   - Tap "Install"
   - Allow "Unknown Sources" if prompted
   - Tap "Open"

### Method 3: ADB (Advanced)

```powershell
# Make sure phone is connected and USB Debugging is enabled
adb install STACK-Kids-Bank.apk
```

## 🔧 Enable "Unknown Sources" (If Needed)

If your phone blocks the installation:

**Android 8.0+:**
1. When you try to install, you'll see a prompt
2. Tap "Settings"
3. Enable "Allow from this source"
4. Go back and install

**Older Android:**
1. Settings → Security
2. Enable "Unknown Sources"
3. Install the APK
