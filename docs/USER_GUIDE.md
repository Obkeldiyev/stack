# STACK Kids Bank - User Guide

## Getting Started

### Creating Your Account

1. **Visit Stack**: Go to the Stack website or open the app
2. **Register**: Click "Get Started" and choose your role:
   - **Parent**: Full control over family finances
   - **Child**: Supervised account with learning features
3. **Verify**: Complete account setup with username and secure password

### Setting Up Your Family

#### For Parents:
1. **Create Family**: Go to Family section and create a new family
2. **Generate Invite**: Click "Invite Members" to get a QR code or invite code
3. **Share Code**: Share the QR code or 6-character code with family members

#### For Children:
1. **Join Family**: Use QR scanner or enter invite code from parent
2. **Account Setup**: Your account will be linked to the family automatically

## Core Features

### Managing Money

#### Sending Allowances
1. Go to **Family** section
2. Select child's account
3. Click **Send Money**
4. Enter amount and optional note
5. Confirm transfer

#### Setting Up Automatic Allowances
1. Go to **Settings** > **Allowances**
2. Select child
3. Set amount and frequency (daily, weekly, monthly)
4. Choose start date
5. Save settings

### Savings Goals

#### Creating Goals (Children)
1. Go to **Goals** section
2. Click **New Goal**
3. Enter goal name and target amount
4. Add optional description
5. Start saving!

#### Saving Money
1. Select your goal
2. Click **Add Money**
3. Choose amount from your account
4. Watch your progress grow

### Educational Games

#### Playing Games
1. Go to **Games** section
2. Choose a game to play
3. Complete the game
4. Earn coins based on performance
5. Coins are added to your account automatically

#### Available Games
- **Math Rush**: Quick math problems
- **Coin Catcher**: Catch falling coins
- **Budget Challenge**: Make smart spending choices
- **Savings Race**: Learn about compound interest
- **Memory Cards**: Match financial terms
- **Number Guess**: Estimate and learn
- **Word Scramble**: Financial vocabulary
- **Smart Quiz**: Financial knowledge test

### Family Management

#### Viewing Family Members
1. Go to **Family** section
2. See all family members and their roles
3. View account balances (parents only)
4. Access member management options

#### Managing Permissions (Parents)
1. Select family member
2. Adjust spending limits
3. Set allowance schedules
4. Monitor transaction history

## Troubleshooting

### Common Issues

#### Can't Join Family
- Check invite code is correct (6 characters)
- Ensure QR scanner has camera permission
- Try manual code entry if QR scan fails
- Contact parent for new invite code

#### Transaction Not Showing
- Wait a few seconds for sync
- Refresh the page/app
- Check internet connection
- Contact support if issue persists

#### Game Not Loading
- Check internet connection
- Clear browser cache
- Update app to latest version
- Try different browser/device

## Tips for Success

### For Parents
- Start with small allowances and increase gradually
- Use transaction notes to teach spending categories
- Review spending patterns together with children
- Set clear rules and expectations
- Celebrate savings milestones

### For Children
- Set realistic savings goals
- Play educational games regularly
- Ask parents about spending decisions
- Track your progress weekly
- Learn from transaction history

## Privacy & Security

### Data Protection
- All data encrypted in transit and at rest

### Account Security

- Enable notifications for all transactions
- Log out from shared devices
- Report suspicious activity immediately
