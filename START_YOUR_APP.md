## 2. 🌐 Web App (Development Mode)

### Run Locally with Backend

**Terminal 1 - Start Backend:**
```bash

mvn spring-boot:run
```
Backend runs on: http://localhost:8080

**Terminal 2 - Start Frontend:**

**Open:** http://localhost:8081 in your browser

## 3. 📱 Android App (2 Options)

### Option A: Progressive Web App (Recommended)

**Step 1: Deploy Online**
```bash
cd stack-family-finance
npm install -g netlify-cli
netlify login
netlify deploy --prod --dir=dist
```

**Step 2: Install on Android**
1. Open the URL on your Android phone
2. Chrome menu (⋮) → "Add to Home Screen"
3. App installs like native app!

**Benefits:**
- ✅ No Android Studio needed
- ✅ Works immediately
- ✅ Auto-updates
- ✅ Full features
- ✅ Looks native

---

### Option B: Native APK (Advanced)

**Requirements:**
- Android Studio installed
- Android SDK configured

**Steps:**
1. Install Android Studio from: https://developer.android.com/studio
2. Open Android Studio
3. Open project: `stack-family-finance\android`
4. Build → Build Bundle(s) / APK(s) → Build APK(s)
5. Wait 2-5 minutes
6. APK location: `android\app\build\outputs\apk\debug\app-debug.apk`
7. Copy to phone and install

**Note:** This requires ~10GB disk space and 20-30 minutes setup time.

## 🔧 Troubleshooting

### Desktop App Won't Start?
- Check Windows Defender didn't block it
- Right-click → Properties → Unblock
- Run as Administrator

### Web App Not Loading?
- Check backend is running (port 8080)
- Check frontend is running (port 8081)
- Clear browser cache

### Android PWA Not Installing?
- Use Chrome browser
- Check internet connection
- Try "Add to Home Screen" again

### APK Build Fails?
- Install Android Studio
- Set ANDROID_HOME variable
- Check SDK is downloaded
