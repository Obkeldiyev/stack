### Step-by-Step Instructions:

#### Step 1: Download Android Studio (15 minutes)

1. **Go to:** https://developer.android.com/studio
2. **Click:** "Download Android Studio"
3. **Run:** The installer
4. **During installation, make sure to install:**
   - Android Studio
   - Android SDK
   - Android SDK Platform-Tools
   - Android Virtual Device (optional)

#### Step 2: Open Your Project (2 minutes)

1. **Launch Android Studio**
2. **Click:** "Open"
3. **Navigate to:** `C:\Users\obkel\OneDrive\Desktop\stack full\stack-family-finance\android`
4. **Click:** "OK"
5. **Wait:** For Gradle sync to complete (2-3 minutes)
   - You'll see "Gradle sync" in the bottom status bar
   - Wait until it says "Gradle sync finished"

#### Step 3: Build APK (5 minutes)

1. **In Android Studio menu, click:**
   - Build → Build Bundle(s) / APK(s) → Build APK(s)

2. **Wait:** 2-5 minutes for build to complete

3. **When done:** You'll see a notification saying "APK(s) generated successfully"

4. **Click:** "locate" in the notification

5. **Your APK is here:**
   ```
   C:\Users\obkel\OneDrive\Desktop\stack full\stack-family-finance\android\app\build\outputs\apk\debug\app-debug.apk
   ```

#### Step 4: Install on Your Phone (2 minutes)

**Method A: USB Cable**
1. Connect your phone to computer via USB
2. Enable "USB Debugging" on your phone:
   - Settings → About Phone
   - Tap "Build Number" 7 times (enables Developer Options)
   - Settings → Developer Options → Enable "USB Debugging"
3. In Android Studio: Run → Run 'app'
4. Select your device
5. App installs automatically!

**Method B: File Transfer**
1. Copy `app-debug.apk` to your phone (via USB, email, Google Drive, etc.)
2. On your phone, open the APK file
3. Tap "Install"
4. If prompted, allow "Install from Unknown Sources"
5. Open "STACK Kids Bank" app

## 🔧 Troubleshooting

### Desktop App Issues:

**App won't start?**
- Right-click → Properties → Unblock
- Run as Administrator
- Check Windows Defender didn't block it

**App crashes?**
- Make sure backend is running (if needed)
- Check the backend URL in settings

### Android Studio Issues:

**"SDK location not found"?**
- Make sure Android Studio is fully installed
- Check that SDK was downloaded during installation
- Restart Android Studio

**Gradle sync fails?**
- File → Invalidate Caches → Invalidate and Restart
- Try again

**Build fails?**
- Build → Clean Project
- Then: Build → Build APK(s)

### APK Installation Issues:

**Can't install APK on phone?**
- Enable "Unknown Sources" in phone settings
- Check phone has enough storage (need ~25MB)
- Try uninstalling old version first

**App crashes on phone?**
- Make sure backend is accessible
- Check internet connection
- Try reinstalling

## 📱 Progressive Web App (PWA) Benefits

### Why PWA is Great for Android:

✅ **No APK needed** - Works without building APK
✅ **No Android Studio** - Saves 10GB disk space
✅ **Auto-updates** - Always latest version
✅ **Cross-platform** - Works on any device
✅ **Smaller size** - ~2MB vs ~25MB APK
✅ **Instant deployment** - 5 minutes vs 30 minutes
✅ **Looks native** - Full-screen, app icon, splash screen
✅ **Works offline** - Can cache for offline use

### How to Install PWA on Android:

1. Deploy web app to Netlify or Vercel (free)
2. Open URL on Android phone in Chrome
3. Chrome menu (⋮) → "Add to Home Screen"
4. Enter app name: "STACK Kids Bank"
5. Tap "Add"
6. **Done!** App appears on home screen like native app
